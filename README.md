Healthcare Analytics and Patient Insight Dashboard 
Project Overview
This project involved the analysis of raw healthcare data first to clean and subsequently establish trends between various data such as demographics, admissions and length of admissions using Power BI, Excel and DAX

Objectives:
. Analyze disease distribution
. Track patient admission trends 
. Evaluate hospital stay duration
. Visualize healthcare billing patterns
. Generate actionable healthcare insights

Tools used:
. Power BI - curated an interactive dashboard with dynamic visualizations
. Microsoft Excel 
. DAX(Data Analysis Expression)

Dashboard Features:
- KPI Cards for patient volume, billingand hospital stay duration
- Disease Distribution analysis
- Monthly admission trends
- Admission type breakdown
- Gender demographic analysis
- Interactive slicers for filtering

DAX Measures used:
Average Billing = AVERAGE(Patients[Billing Amount])
Average Stay Duration = AVERAGE(Patients[Length of Stay])

Key Insights:
- Diabetes was the most common medical condition
- Average patient stay duration was approximately 16 days
- Emergency admission accounted for a large portion of visits
- Healthcare billing showed significant fluctuation based on condition

Files Included:
- Power BI dashboard(.pbix)
- Cleaned healthcare dataset (.xlsx)
- Dashboard preview image
